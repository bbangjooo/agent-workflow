```tsx

  title: '서비스명',
  description: '서비스 설명',
  keywords: ['키워드1', '키워드2'],
}
```

- [ ] 제목 (title) 설정

### Open Graph
```tsx
openGraph: {
  title: '서비스명',
  description: '서비스 설명',
  url: 'https://example.com',
  siteName: '서비스명',
  images: [
    {
      url: 'https://example.com/og-image.png',
      width: 1200,
      height: 630,
    },
  ],
  locale: 'ko_KR',
  type: 'website',
}
```

- [ ] OG 이미지 생성 (1200x630)
- [ ] OG 메타 태그 설정
