# Write Meta (Facebook/Instagram) Launch

Meta 플랫폼(Facebook, Instagram)용 런칭 콘텐츠를 생성합니다.

## 설명

Facebook 페이지 포스트, Instagram 피드/릴스/스토리 콘텐츠, Meta Ads 크리에이티브 카피를 생성합니다. 바로 복붙해서 올릴 수 있는 형태로 출력합니다.

## 입력

> **참고**: 아래 리서치 산출물이 존재하면 반드시 참조하여 콘텐츠를 생성합니다:
> - `outputs/promotion/copy-research.md` — 경쟁사 카피 패턴 → 검증된 구조 활용
> - `outputs/promotion/channel-research.md` — 커뮤니티 규칙/인기글 패턴 → 규칙 준수 & 효과적 포맷
> - `outputs/promotion/audience-language.md` — 유저 실제 표현 → 유저 언어로 작성, 금지 용어 회피
> - `outputs/promotion/timing-research.md` — 최적 포스팅 시간 → 산출물에 추천 시간 명시

- 제품명, 한줄 설명, 핵심 기능, 빌딩 스토리, 라이브 URL
- 타겟 오디언스 (연령대, 관심사)
- 비주얼 에셋 유무 (스크린샷, 제품 이미지, 영상)

## 실행 내용

### 1. Facebook 런칭 포스트 (3개 시안)

```
시안 A (스토리텔링형):
- 문제 → 해결 스토리 구조
- 개인적 빌딩 경험 포함
- 1,000-1,500자 (긴 포스트가 참여율 높음)
- CTA: 댓글로 의견 요청 또는 링크 클릭 유도
- 이미지: 제품 스크린샷 또는 비포/애프터

시안 B (가치 제안형):
- 핵심 혜택 3가지 불릿 포인트
- 소셜 프루프 포함 (사용자 수, 피드백 등)
- 500-800자
- CTA: "지금 무료로 시작하기" + 링크
- 이미지: 기능 하이라이트 카드

시안 C (질문/공감형):
- 타겟 오디언스의 고충으로 시작하는 질문
- 공감 → 해결책으로 자연스럽게 연결
- 300-500자
- CTA: 댓글 참여 유도
- 이미지: 문제-해결 비주얼
```

### 2. Instagram 피드 포스트 (캐러셀 + 단일)

```
[캐러셀 포스트 — 5-10 슬라이드]
슬라이드 1 (훅): 강렬한 헤드라인 + 시선 끄는 비주얼
슬라이드 2: 문제 제시 ("이런 경험 있지 않나요?")
슬라이드 3-6: 핵심 기능별 소개 (기능 + 혜택)
슬라이드 7: 사용 전/후 비교
슬라이드 8: 가격/무료 플랜 안내
슬라이드 9: CTA ("링크는 바이오에!")

캡션:
- 첫 줄에 훅 (더보기 클릭 유도)
- 핵심 내용 요약 (150-200자)
- CTA + 바이오 링크 안내
- 해시태그 20-30개 (첫 댓글에 분리 가능)

[단일 이미지 포스트 — 3개 시안]
- 제품 스크린샷 + 짧은 카피
- 사용자 후기/결과물 쇼케이스
- 빌드인퍼블릭 업데이트
```

### 3. Instagram 릴스 스크립트 (3개)

```
릴스 A (문제-해결 — 15-30초):
- [0-3초] 훅: "{문제 상황} 겪어본 적 있나요?"
- [3-10초] 문제 시연 (기존 방식의 불편함)
- [10-20초] 해결: 제품으로 해결하는 모습
- [20-25초] 결과: 비포/애프터
- [25-30초] CTA: "바이오 링크에서 무료로 시작"
- 음악: 트렌딩 오디오 추천
- 텍스트 오버레이 포함

릴스 B (튜토리얼 — 30-60초):
- [0-3초] 훅: "{결과}를 {N}초 만에 하는 법"
- [3-45초] 단계별 사용법 시연
- [45-55초] 최종 결과물
- [55-60초] CTA
- 자막 필수 (무음 시청 대비)

릴스 C (트렌드 활용 — 15초):
- 트렌딩 밈/포맷 활용
- 제품을 자연스럽게 녹여내기
- 엔터테인먼트 + 정보 균형
```

### 4. Instagram 스토리 시퀀스 (런칭 당일)

```
스토리 1: 카운트다운 스티커 ("드디어 런칭!")
스토리 2: 제품 티저 영상/이미지
스토리 3: 핵심 기능 소개 (설문 스티커: "이 기능 써보고 싶은 사람?")
스토리 4: 링크 스티커로 제품 페이지 연결
스토리 5: Q&A 스티커 ("궁금한 거 물어보세요!")
스토리 6: 사용자 반응/피드백 공유
```

### 5. Meta Ads 크리에이티브 카피 (3세트)

```
[광고 세트 A — 전환 캠페인]
프라이머리 텍스트: {문제} → {해결책} (125자 이내)
헤드라인: {핵심 혜택} (40자 이내)
설명: {부가 혜택 또는 소셜 프루프} (30자 이내)
CTA 버튼: "자세히 알아보기" / "무료 체험"
타겟: {관심사 타겟팅 추천}

[광고 세트 B — 리타겟팅]
프라이머리 텍스트: "아직 시작 안 하셨나요?" + 혜택 리마인드
헤드라인: {긴급성 또는 한정 오퍼}
CTA 버튼: "지금 시작하기"

[광고 세트 C — 유사 타겟]
프라이머리 텍스트: 사회적 증거 중심 (N명이 사용 중)
헤드라인: {카테고리 리더십 강조}
CTA 버튼: "무료로 시작"
```

### 6. 해시태그 전략

```
[Instagram]
- 대형 해시태그 (100만+): 5-7개 — 노출 극대화
- 중형 해시태그 (1만-100만): 10-15개 — 경쟁/노출 균형
- 소형/니치 해시태그 (1만 미만): 5-8개 — 타겟 커뮤니티 침투
- 브랜드 해시태그 1개
+ 제품 도메인별 해시태그

[Facebook]
- 해시태그 3-5개 (과도한 사용 지양)
- 그룹/페이지 관련 태그 위주
```
