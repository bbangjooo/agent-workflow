# Write LinkedIn Post

LinkedIn 런칭 포스트와 시리즈 콘텐츠를 생성합니다.

## 설명

LinkedIn 피드에 맞는 전문가 톤의 런칭 포스트, 빌딩 스토리, Thought Leadership 포스트를 생성합니다. B2B/프로페셔널 오디언스를 타겟으로 하며, 바로 복붙해서 올릴 수 있는 형태로 출력합니다.

## 입력

> **참고**: 아래 리서치 산출물이 존재하면 반드시 참조하여 콘텐츠를 생성합니다:
> - `outputs/promotion/copy-research.md` — 경쟁사 카피 패턴 → 검증된 구조 활용
> - `outputs/promotion/channel-research.md` — 커뮤니티 규칙/인기글 패턴 → 규칙 준수 & 효과적 포맷
> - `outputs/promotion/audience-language.md` — 유저 실제 표현 → 유저 언어로 작성, 금지 용어 회피
> - `outputs/promotion/timing-research.md` — 최적 포스팅 시간 → 산출물에 추천 시간 명시

- 제품명, 한줄 설명, 핵심 기능, 빌딩 스토리, 라이브 URL
- 창업자/팀 프로필 (직함, 경력 배경)
- 타겟 업계/직무 (예: PM, 개발자, 마케터, 스타트업 대표)

## 실행 내용

### Step 1. LinkedIn 채널 리서치

> 이 단계의 산출물(`outputs/promotion/linkedin-research.md`)이 이미 존재하면 스킵합니다.

LinkedIn에서 효과적인 포스트를 작성하기 위해 플랫폼 특성과 트렌드를 조사합니다.

```
1. LinkedIn 알고리즘 & 포맷 특성 조사
   - 피드 알고리즘 우선순위 (댓글 > 좋아요 > 공유)
   - 최적 글자수 범위 (1,300자 훅 컷오프, 총 3,000자 권장)
   - 외부 링크 포함 시 도달률 패널티 여부
   - 이미지/카루셀/동영상/텍스트 전용 포스트별 참여율

2. 업계별 인기 포스트 패턴 분석
   - 타겟 업계 LinkedIn 인플루언서 5-10명 식별
   - 최근 인기 포스트 패턴 분석 (구조, 톤, 길이, 훅)
   - 바이럴 포스트 공통 요소 추출
   - 제품 카테고리 관련 인기 해시태그

3. 경쟁사/유사 제품 LinkedIn 활동 분석
   - 경쟁사 회사 페이지 & 창업자 개인 계정 포스트
   - 어떤 포스트가 높은 참여를 받았는지
   - 포스팅 빈도 & 시리즈 전략

4. LinkedIn 셀프프로모 Best Practice
   - "런칭 포스트" vs "가치 제공 포스트" 비율 (80/20 규칙)
   - 댓글 유도 전략 (질문형 CTA)
   - 외부 링크는 첫 댓글에 배치 (알고리즘 우회)
   - 태그/멘션 전략
```

### Step 2. 런칭 포스트 (3개 시안)

```
시안 A (빌딩 스토리형 — 고참여):
- 첫 줄 훅: 개인 경험 또는 놀라운 사실 (스크롤 멈추게)
- 문제 발견 경위 (1인칭 스토리)
- 해결 과정 & 배운 점
- 제품 소개 (자연스러운 전환)
- CTA: 댓글로 의견 요청
- 링크는 첫 댓글에 배치
- 1,500-2,500자

시안 B (문제-해결형 — 가치 제공):
- 첫 줄 훅: 타겟 오디언스의 고충점
- 문제 구체화 (데이터/사례)
- 해결 접근법 설명 (교육적 가치)
- 제품을 해결책으로 자연스럽게 소개
- CTA: "비슷한 경험 있으신 분?"
- 1,000-1,500자

시안 C (성과/마일스톤형):
- 첫 줄 훅: 구체적 숫자/성과
- 빌딩 타임라인 (단계별)
- 핵심 인사이트 3가지
- 제품 링크 & 다음 계획
- CTA: "어떤 기능이 가장 끌리시나요?"
- 800-1,200자
```

### Step 3. Thought Leadership 시리즈 (5편 계획)

```
[시리즈 테마: 제품이 속한 도메인의 인사이트]

1편: 업계 문제 진단
- 왜 이 문제가 아직 해결 안 됐는지
- 데이터/사례 기반 분석
- 제품 직접 언급 없이 문제만 다룸

2편: 기존 해결책의 한계
- 현재 대안들의 구조적 문제
- 실제 사용 경험 기반 비교
- 제품 간접 언급 (가볍게)

3편: 새로운 접근법 제시
- 제품의 철학/접근법 소개
- 기술적 차별점 (비전문가도 이해 가능하게)
- 사례/데모 포함

4편: 빌딩 과정 비하인드
- 기술적/사업적 도전과 극복
- 실패한 시도 & 배운 점
- 개발자/창업자 커뮤니티에 공감

5편: 비전 & 로드맵
- 제품의 미래 방향
- 업계 트렌드와 연결
- 피드백 요청 & 커뮤니티 초대
```

### Step 4. 개별 프로모 포스트 (5개 시안)

```
[기능 하이라이트]
- 특정 기능 하나에 집중
- 사용 전/후 비교 (가능하면 스크린샷)
- 짧고 임팩트 있게 (500-800자)

[유저 성공 사례]
- 초기 유저 피드백/결과 공유
- 인용 + 감사 표현
- 소셜 프루프 효과

[업계 인사이트 + 제품 연결]
- 트렌드/뉴스 → 제품 연관성
- 교육적 가치 먼저, 프로모 나중에
- 해시태그 전략적 사용

[비하인드 더 씬]
- 개발/디자인 과정 공유
- 어려웠던 결정 & 선택 이유
- 빌드인퍼블릭 톤

[질문/토론 유도]
- 업계 관련 오픈 질문
- 제품과 관련된 주제로 자연스럽게 연결
- 댓글 참여 극대화
```

### Step 5. LinkedIn 카루셀 (문서/슬라이드형)

```
[카루셀 — 8-12 슬라이드]
슬라이드 1 (표지): 강렬한 헤드라인 + 서브카피
슬라이드 2: 문제 제시
슬라이드 3-7: 핵심 포인트 (1 슬라이드 = 1 포인트)
슬라이드 8: 제품 소개
슬라이드 9: CTA + 링크 안내

디자인 가이드:
- 슬라이드당 텍스트 30단어 이내
- 큰 폰트, 대비 높은 색상
- 브랜드 컬러 일관 사용
- PDF 형태로 업로드
```

### Step 6. 해시태그 전략

```
- LinkedIn 해시태그는 3-5개가 최적 (과다 사용 지양)
- 업계 해시태그 2-3개 + 일반 해시태그 1-2개
- 브랜드 해시태그 1개 (있으면)
- 인기 해시태그: #startup #buildinpublic #SaaS #productivity #AI
+ 제품 도메인별 해시태그
```
