# Write Naver Blog

네이버 블로그 SEO 최적화 포스트를 생성합니다.

## 설명

네이버 검색 노출에 최적화된 블로그 포스트를 작성합니다. 네이버 SEO 알고리즘(C-Rank, D.I.A.)에 맞춘 구조, 키워드, 길이를 준수합니다.

## 입력

> **참고**: `outputs/promotion/copy-research.md`가 존재하면 경쟁사/성공 사례의 실제 카피 패턴을 참조하여 콘텐츠를 생성합니다. 검증된 패턴을 우리 제품에 맞게 변형합니다.

- 제품명, 핵심 기능, 타겟 유저, 해결하는 문제, 라이브 URL

## 실행 내용

### 1. 키워드 전략

```
메인 키워드: {제품이 해결하는 문제} 관련 검색어
서브 키워드: {제품 카테고리} + {타겟 유저 니즈}

예시:
- 메인: "프리랜서 송장 관리"
- 서브: "프리랜서 세금 관리 앱", "인보이스 자동화", "프리랜서 필수 앱"

키워드 배치:
- 제목에 메인 키워드 포함
- 본문 첫 100자 내 메인 키워드
- 소제목(H2)에 서브 키워드
- 본문 전체에 자연스럽게 3-5회 반복
```

### 2. 포스트 구조

```
[제목] — 메인 키워드 포함, 30자 내외
예: "{문제} 해결하는 {제품명} 사용 후기"
예: "{타겟유저}를 위한 {카테고리} 추천 — {제품명}"

[본문] — 1,500자 이상 (네이버 SEO 최소 권장)

구조:
1. 도입 (200자): 문제 공감 + 해결책 암시
2. 제품 소개 (300자): 무엇인지, 누가 만들었는지
3. 핵심 기능 (500자): 기능별 스크린샷 + 설명
4. 사용 후기/장점 (300자): 실제 사용 경험 톤
5. 단점/아쉬운 점 (100자): 신뢰도 확보
6. 마무리 (100자): CTA + 링크

톤: 경험 공유 톤 (광고 아닌 후기 느낌)
이미지: 3장 이상 (네이버 SEO 가점)
```

### 3. 네이버 SEO 체크리스트

```
- [ ] 제목에 메인 키워드 포함
- [ ] 본문 1,500자 이상
- [ ] 이미지 3장 이상 (ALT 텍스트 포함)
- [ ] 소제목 2-3개 사용
- [ ] 태그 10개 설정
- [ ] 카테고리 적절히 설정
- [ ] 링크는 본문 하단에 자연스럽게 배치
```

### 4. 시리즈 포스트 계획

```
단일 포스트가 아닌, 시리즈로 지속 노출:
1편: "{문제} 해결하는 방법 총정리" (일반 정보 + 제품 자연스럽게)
2편: "{제품명} 사용법 완전 가이드" (튜토리얼)
3편: "{제품명} vs {경쟁사} 비교" (비교 콘텐츠)
4편: "{타겟유저} 필수 도구 N선" (리스트 콘텐츠)
```

## 산출물

`outputs/promotion/naver-blog/blog-post.md`

```markdown
# 네이버 블로그 포스트

## SEO 정보
- **메인 키워드**: {키워드}
- **서브 키워드**: {키워드 목록}
- **태그**: {태그 10개}
- **카테고리**: {카테고리}

## 제목
{제목}

## 본문

{전체 본문 — 1,500자 이상, 바로 복붙 가능}

{이미지 삽입 위치 표시: [이미지: {설명}]}

## 시리즈 계획

| 회차 | 제목 | 키워드 | 발행 시기 |
|------|------|--------|----------|
| 1편 | {제목} | {키워드} | 런칭 전 |
| 2편 | {제목} | {키워드} | 런칭 주간 |
| 3편 | {제목} | {키워드} | 런칭 +1주 |
| 4편 | {제목} | {키워드} | 런칭 +2주 |
```

## 완료 조건

- 포스트 본문 1,500자 이상
- 메인/서브 키워드 전략 포함
- 이미지 삽입 위치 3곳 이상 지정
- 태그 10개 설정
- 시리즈 포스트 4편 계획
