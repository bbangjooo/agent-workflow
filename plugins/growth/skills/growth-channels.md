# Growth Channels

Step 5.4: 그로스 채널 설계

## 설명

지속적 성장을 위한 채널과 메커니즘을 설계하는 스킬입니다. PLG, 레퍼럴, 유료 광고, 파트너십 전략을 포함합니다.

## 트리거

- Step 5.3 (Launch Plan) 완료 후

## 입력

- `outputs/stage-5/market-analysis.md` — 타겟 시장 및 페르소나
- `outputs/stage-4/deployment-complete.md` — 배포 정보

## 실행 내용

### 1. Product-Led Growth (PLG) 설계

```
- 프리미엄 모델 경계 설정
  - 무료 tier에서 제공할 핵심 가치
  - 유료 전환 트리거 (사용량, 기능, 팀 규모)
  - 목표: 첫 "와우 모먼트" 5분 이내 도달

- 바이럴 루프 메커니즘
  - 공유 가능한 산출물 (워터마크, 링크백)
  - "팀원 초대" 기능
  - UGC (사용자 생성 콘텐츠) 연동

- 온보딩 최적화
  - 셀프서브 온보딩 플로우 설계
  - 핵심 가치 도달까지의 단계 수 최소화
```

### 2. 레퍼럴 프로그램

```
- 양측 보상 구조 (추천인 + 피추천인)
- 보상 유형 결정
  - 제품 관련 보상 (추가 기능, 용량) — 권장
  - 금전 보상 (크레딧, 할인)
- 인앱 추천 UX 흐름
  - 초대 링크 생성 → 공유 → 가입 → 보상 지급
- 도구: Rewardful, Viral Loops 등
```

### 3. 유료 광고 테스트

```
[국내]
- 네이버 검색 광고 (브랜드 키워드 + 관련 키워드)
- 카카오 비즈보드 (카카오톡 내 디스플레이)
- 소액 테스트: 채널당 일 1-3만원으로 시작

[글로벌]
- Google Ads (검색 + 디스플레이)
- Meta Ads (Facebook/Instagram — 소비자 앱)
- Apple Search Ads (모바일 앱)
- LinkedIn Ads (B2B SaaS)
- 소액 테스트: 채널당 일 $10-30으로 시작

- A/B 테스트 계획
  - 크리에이티브 2-3개 시안
  - 타겟 오디언스 세그먼트
  - 전환 이벤트 정의
```

### 4. 파트너십/통합 전략

```
- 플랫폼 마켓플레이스 등록
  - Slack App Directory, Zapier, Notion 등
  - [국내] 카카오 i, 네이버 클로바 등
- 크로스 프로모션 대상 서비스 리스트
- 어필리에이트 프로그램 설계 (전환당 수수료)
```

### 5. ASO (모바일 앱인 경우)

```
- 키워드 전략 (롱테일, 의도 기반)
- 스크린샷 및 프리뷰 영상 계획
- 리뷰 관리 전략 (인앱 리뷰 요청 타이밍)
- [국내] ONE Store 추가 등록
```

### 6. 이벤트/컨퍼런스

```
[국내] COMEUP, Startup Grind Seoul, 액셀러레이터 데모데이
[글로벌] TechCrunch Disrupt, Web Summit, SXSW
- 참가 목적 (네트워킹 / 피칭 / 부스)
- 비용 대비 효과 평가
```
