# Launch Plan

Step 5.3: 런칭 플랜 작성

## 설명

출시 주간의 상세 실행 계획을 작성하는 스킬입니다. 플랫폼별 런칭 전략, PR, 인플루언서 시딩을 포함합니다.

## 트리거

- Step 5.2 (Pre-launch Strategy) 완료 후

## 입력

- `outputs/stage-5/market-analysis.md` — 타겟 시장 및 페르소나
- `outputs/stage-5/pre-launch-strategy.md` — 프리런칭 전략

## 실행 내용

### 1. 런칭 플랫폼 전략

```
[국내]
- 디스콰이엇 런칭
  - 제품 페이지 작성 (스크린샷, 데모 영상)
  - 메이커 프로필 최적화
  - 런칭 데이 커뮤니티 활성화

[글로벌]
- Product Hunt 런칭 플레이북
  - 타이틀 3개 시안 (가치/소셜프루프/호기심 앵글)
  - 60-70자, 고객 중심 메시지
  - 런칭 요일/시간 결정 (화-목 추천, 첫 4시간 + 마지막 2시간 집중)
  - 팔로워 사전 확보 (런칭 2-4주 전)
  - 시간대별 실행 체크리스트
- Show HN 포스팅
  - "Show HN: {제품명} — {한줄 설명}" 포맷
  - 기술적 질문 대응 준비
- Reddit 포스팅
  - 관련 서브레딧 리스트 (r/startups, r/SaaS 등)
  - "I built..." 프레이밍 사용
  - 가치 제공 우선, 제품 소개는 자연스럽게
```

### 2. PR/언론 전략

```
[국내]
- 보도자료 작성 (벤처스퀘어, 플래텀, 아웃스탠딩, 모비인사이드)
- 기자 리스트 & 관계 구축 (런칭 2-3개월 전부터)

[글로벌]
- TechCrunch Tips 제출
- 테크 블로거/뉴스레터 아웃리치
- 기자의 최근 기사 5-10개 읽고 맞춤 피칭
```

### 3. 인플루언서 시딩

```
- 마이크로 인플루언서 (1만-10만 팔로워) 우선 타겟
  - 매크로 대비 30% 높은 ROI, 7-20% 참여율
- 얼리 액세스 + 독점 기능 제공

[국내] 유튜버, 네이버 블로거 타겟 리스트
[글로벌] Twitter/X, YouTube 크리에이터 타겟 리스트
```

### 4. 커뮤니티 동시 포스팅

```
[국내] 네이버 카페, GeekNews, OKKY, Velog
[글로벌] IndieHackers, Dev.to, 관련 Slack/Discord
- 각 커뮤니티 톤에 맞는 포스트 작성
- 런칭 데이 ± 1일 이내 포스팅
```

### 5. 웨이트리스트 활성화

```
- 초대 이메일 시리즈 설계 (3통)
  - 1통: "드디어 출시!" + 접속 링크
  - 2통: "첫 유저 가이드" + 핵심 기능 안내
  - 3통: "친구 초대하기" + 레퍼럴 링크
```
