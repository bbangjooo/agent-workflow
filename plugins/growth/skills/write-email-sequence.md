# Write Email Sequence

웨이트리스트 활성화 및 온보딩 이메일 시퀀스를 생성합니다.

## 설명

웨이트리스트 유저를 활성 유저로 전환하는 이메일 시리즈와, 신규 가입자 온보딩 이메일 시리즈를 생성합니다. 각 이메일은 제목, 프리헤더, 본문까지 바로 사용 가능한 형태로 출력합니다.

## 입력

> **참고**: `outputs/promotion/copy-research.md`가 존재하면 경쟁사/성공 사례의 실제 카피 패턴을 참조하여 콘텐츠를 생성합니다. 검증된 패턴을 우리 제품에 맞게 변형합니다.

- 제품명, 핵심 가치, 핵심 기능 3-5개, 라이브 URL, 가격 모델

## 실행 내용

### 1. 웨이트리스트 활성화 시퀀스 (3통)

```
[이메일 1: 런칭 알림] — 런칭 당일
제목: "드디어! {제품명}이 출시되었습니다 🎉"
프리헤더: "대기해주신 감사 + 특별 혜택"
본문:
- 감사 인사 (일찍 관심 가져준 분)
- 제품 한줄 소개
- CTA 버튼: "지금 시작하기"
- 얼리 액세스 특별 혜택 (있으면)

[이메일 2: 가치 전달] — 런칭 +2일
제목: "{제품명}으로 {핵심 가치}하는 방법"
프리헤더: "3분 만에 시작하기"
본문:
- 핵심 기능 1개 딥다이브
- 스크린샷/GIF 포함
- "이렇게 하면 됩니다" 스텝 가이드
- CTA 버튼: "직접 해보기"

[이메일 3: 레퍼럴] — 런칭 +5일
제목: "친구에게 {제품명}을 소개해주세요"
프리헤더: "추천하면 {보상} 드려요"
본문:
- 초기 유저 감사
- 레퍼럴 프로그램 소개 (보상 설명)
- 고유 추천 링크
- CTA 버튼: "친구 초대하기"
```

### 2. 온보딩 시퀀스 (5통)

```
[이메일 1: 환영] — 가입 즉시
제목: "{제품명}에 오신 것을 환영합니다!"
본문:
- 환영 + 가입 축하
- "첫 번째로 해볼 것" 1가지 (핵심 가치 최단 경로)
- CTA: "시작하기"

[이메일 2: 핵심 기능] — 가입 +1일
제목: "{핵심기능1} 써보셨나요?"
본문: 가장 중요한 기능 튜토리얼

[이메일 3: 팁] — 가입 +3일
제목: "{제품명} 200% 활용하는 팁 3가지"
본문: 파워 유저 팁

[이메일 4: 소셜 프루프] — 가입 +5일
제목: "다른 {타겟유저}들은 이렇게 사용해요"
본문: 사용 사례 / 후기 (있으면)

[이메일 5: 업그레이드] — 가입 +7일 (프리미엄 모델 시)
제목: "Pro로 업그레이드하면 달라지는 것들"
본문: 유료 기능 소개 + 특별 할인
```

### 3. 이메일 작성 원칙

```
- 제목: 40자 이내, 이모지 1개 이하
- 프리헤더: 60자 이내
- 본문: 150-300단어 (모바일 최적화)
- CTA 버튼: 1개 (명확한 액션)
- 발신자: 사람 이름 + 제품명 (예: "지수 from {제품명}")
- 톤: 친근하고 도움이 되는 (세일즈 아닌 가이드)
```
