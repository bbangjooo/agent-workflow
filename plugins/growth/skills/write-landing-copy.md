# Write Landing Copy

랜딩 페이지 / 웨이트리스트 페이지 카피를 생성합니다.

## 설명

전환율 최적화된 랜딩 페이지 카피를 섹션별로 작성합니다. 헤드라인부터 CTA까지, 개발자가 바로 구현할 수 있는 형태로 모든 텍스트를 출력합니다.

## 입력

- 제품명, 핵심 가치 제안, 타겟 유저, 핵심 기능 3-5개, 가격 모델, 라이브 URL

## 실행 내용

### 1. Hero Section

```
[헤드라인] — 10단어 이내, 핵심 가치
  3개 시안:
  - 결과 중심: "{결과}를 {시간}만에"
  - 문제 중심: "더 이상 {문제}에 시간 낭비하지 마세요"
  - 대상 중심: "{타겟}을 위한 {카테고리}"

[서브헤드라인] — 20단어 이내, 보충 설명
  헤드라인이 What이면 서브는 How
  헤드라인이 How이면 서브는 What

[CTA 버튼] — 3-5단어
  - "무료로 시작하기"
  - "지금 시작하기 — 무료"
  - "{핵심가치} 시작하기"

[소셜 프루프] (있으면)
  - "N명의 {타겟}이 사용 중"
  - "Product Hunt #1"
```

### 2. Problem Section

```
[섹션 제목] — 문제 공감

[문제 포인트 3개]
각각: 아이콘 + 제목 (5단어) + 설명 (20단어)
- 😤 {문제1 제목}: {설명}
- ⏰ {문제2 제목}: {설명}
- 💸 {문제3 제목}: {설명}
```

### 3. Solution / Features Section

```
[섹션 제목] — 해결책 제시

[기능 카드 3-5개]
각각: 제목 + 설명 (2문장) + 스크린샷 지정
- {기능1 제목}: {설명}. {결과}.
  📸 [스크린샷: {화면 설명}]
```

### 4. How It Works

```
[섹션 제목] — "3단계로 시작하세요" 등

Step 1: {동작} — {설명}
Step 2: {동작} — {설명}
Step 3: {동작} — {설명}
```

### 5. Social Proof / Testimonials

```
[섹션 제목]

[후기 카드] (있으면)
각각: 인용문 + 이름 + 직함/회사

[로고 배너] (있으면)
"XX, YY, ZZ가 사용합니다"

[없으면]
- 메이커 스토리로 대체
- 베타 유저 피드백으로 대체
```

### 6. Pricing Section (해당 시)

```
[섹션 제목] — "심플한 가격"

[가격 카드]
business-model.md 기반 티어별 카드:
- 플랜명
- 가격 (월/연)
- 포함 기능 (체크마크 리스트)
- CTA 버튼
- "추천" 배지 (메인 티어)
```

### 7. FAQ Section

```
Q&A 5-7개:
- "무료인가요?" / 가격 관련
- "어떤 {타겟}에게 적합한가요?"
- "{경쟁사}와 뭐가 다른가요?"
- "데이터는 안전한가요?"
- "취소는 쉬운가요?"
```

### 8. Final CTA

```
[헤드라인] — Hero와 다른 앵글
[서브헤드라인]
[CTA 버튼] — Hero와 동일
```

## Social Proof

- 전체 8개 섹션 카피 완성
- 각 섹션 바로 구현 가능한 텍스트
- SEO 메타 태그 포함
