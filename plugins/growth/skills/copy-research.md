# Copy Research

경쟁사 및 성공 사례의 홍보 문구를 실제 조사하여 분석합니다.

## 설명

경쟁사와 유사 성공 서비스들이 **실제로 어떤 문구로 홍보하고 있는지** 웹 서치를 통해 조사합니다. 랜딩 페이지 헤드라인, Product Hunt 타이틀, 소셜미디어 포스트, 앱스토어 설명 등 채널별 실제 카피를 수집하고 패턴을 분석합니다.

> `/promote`로 콘텐츠 생성 시 이 조사 결과를 참고하여, 검증된 카피 패턴을 우리 제품에 맞게 변형합니다.

## 트리거

- `/promote` 실행 시 자동 선행
- `/grow` Step 6.3 (Launch Plan) 진행 시

## 입력

- `outputs/stage-5/market-research.md` — 경쟁사 목록, 커뮤니티 목록
- 프로젝트 컨텍스트 (제품명, 카테고리, 타겟 유저)

## 실행 내용

### 1. 랜딩 페이지 카피 조사 (웹 서치 → 사이트 방문)

```
경쟁사 + 유사 성공 서비스 랜딩 페이지에서 수집:

[수집 항목]
- 헤드라인 (H1)
- 서브헤드라인
- CTA 버튼 텍스트
- 가치 제안 (Value Proposition) 섹션 문구
- 소셜 프루프 문구 ("N명이 사용 중", "XX 기업이 신뢰하는")
- FAQ 항목 (어떤 질문에 답하고 있는지)

[분석]
- 공통 패턴: 어떤 구조가 반복되는가
- 차별화 문구: 경쟁사가 뭘 강조하는가
- 우리가 활용할 수 있는 앵글
```

### 2. Product Hunt 카피 조사 (웹 서치)

```
[조사 방법]
- 웹 서치로 "{카테고리} site:producthunt.com" 검색
- 상위 제품 5개의 PH 페이지 확인

[수집 항목]
- 타이틀 (tagline)
- 서브타이틀
- 첫 번째 댓글 (메이커 코멘트)
- 투표 수 (인기도 참고)

[분석]
- 높은 투표를 받은 제품의 타이틀 패턴
- 메이커 코멘트의 공통 구조
- 어떤 앵글이 반응이 좋았는지
```

### 3. 소셜미디어/커뮤니티 카피 조사 (웹 서치)

```
[Twitter/X]
- 웹 서치로 "{경쟁사} launch twitter" 검색
- 경쟁사의 런칭 트윗/스레드 수집
- 어떤 표현이 RT/좋아요를 많이 받았는지

[Reddit]
- 웹 서치로 "{카테고리} site:reddit.com I built" 검색
- 높은 upvote를 받은 포스트의 제목 패턴
- 본문 구조 (스토리텔링 방식)

[IndieHackers]
- 웹 서치로 "{카테고리} site:indiehackers.com" 검색
- "How I got first 100 users" 류 포스트에서 문구 패턴

[한국]
- 웹 서치로 "{카테고리} 디스콰이엇" 검색
- 웹 서치로 "{카테고리} 네이버 블로그" 검색
- 한국어 카피 패턴 수집
```

### 4. 앱스토어 카피 조사 (모바일 제품 시)

```
[수집 항목]
- 앱 이름 + 부제
- 스크린샷 캡션 텍스트
- 앱 설명 첫 3줄 (fold 위)
- 주요 키워드

[분석]
- 어떤 키워드를 타이틀에 넣는지
- 스크린샷 캡션의 패턴
- 설명에서 기능 vs 혜택 비중
```

### 5. 이메일/뉴스레터 카피 조사

```
[조사 방법]
- 경쟁사 서비스에 실제 가입하여 환영 이메일 확인 (가능 시)
- 웹 서치로 "{경쟁사} welcome email" 검색
- 웹 서치로 "{경쟁사} onboarding email" 검색

[수집 항목]
- 환영 이메일 제목 + 구조
- 온보딩 시퀀스 패턴 (몇 통, 간격, 내용)
- CTA 문구
```

### 6. 카피 패턴 종합 분석

```
수집된 문구를 분석하여:

[헤드라인 패턴]
- 결과 중심: "X를 Y 만에" (Notion: "Write, plan, organize")
- 문제 중심: "더 이상 X하지 마세요"
- 대상 중심: "{타겟}을 위한 {카테고리}"
- 비교 중심: "X의 대안" ("Calendly alternative")

[CTA 패턴]
- "무료로 시작하기" / "Get started free"
- "Try for free" / "Start free trial"
- "{핵심동작} 시작하기"

[소셜 프루프 패턴]
- 숫자 기반: "10,000+ teams trust us"
- 로고 기반: "Used by teams at Google, Meta..."
- 리뷰 기반: "4.8/5 on G2"

[감정 유발 패턴]
- 공감: "We've all been there..."
- 호기심: "What if you could..."
- 긴급함: "Join 500+ founders who already..."
```

- 소셜미디어/커뮤니티 인기 포스트 카피 최소 5개 수집
- 패턴 분석 + 우리 제품 적용 가이드 작성
- `copy-research.md` 파일 생성

## 사용처

- `/promote` 실행 시 모든 write-* 스킬이 이 파일을 참조
