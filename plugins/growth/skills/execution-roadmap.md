## 실행 내용

### 1. 12주 타임라인 로드맵

```
Week 1-2: 프리런칭 준비
- 웨이트리스트 랜딩페이지 구축
- 커뮤니티 채널 개설
- 콘텐츠 발행 시작

Week 3-4: 프리런칭 가속
- 베타 테스터 모집 시작
- 인플루언서 시딩 준비
- PR 기자 관계 구축
- 콘텐츠/SEO 지속 발행

Week 5-6: 런칭
- 메인 런칭 (Product Hunt / 디스콰이엇)
- Show HN, Reddit 포스팅
- PR 보도자료 배포
- 커뮤니티 동시 포스팅
- 웨이트리스트 활성화

Week 7-8: 포스트런칭 1
- 레퍼럴 프로그램 활성화
- 유료 광고 소액 테스트 시작
- 유저 피드백 수집 및 반영
- 온보딩 최적화

Week 9-10: 포스트런칭 2
- 광고 채널 A/B 테스트 분석
- 효과적 채널 스케일업
- 파트너십/통합 추진
- PLG 메커니즘 최적화

Week 11-12: 최적화
- 전체 채널 성과 분석
- ROI 기반 예산 재배분
- 2차 런칭 (업데이트/신기능) 계획
- 다음 분기 전략 수립
```

### 2. KPI/측정 지표 정의

```
핵심 지표 (North Star):
- DAU/WAU/MAU

획득 지표:
- 웨이트리스트 가입자 수
- 신규 가입자 수 (일/주)
- 채널별 유입량
- 가입 전환율

성장 지표:
- 레퍼럴 계수 (viral coefficient)
- 초대 전환율
- NPS (순추천지수)

비용 지표:
- CAC (고객 획득 비용)
- 채널별 CPA
- LTV/CAC 비율

참여 지표:
- 온보딩 완료율
- 핵심 기능 사용률
- Day 1/7/30 리텐션
```

### 3. 채널별 우선순위 매트릭스

```
이전 Step들의 결과를 종합하여:
- 1순위 (즉시 실행): Impact 높고 + Effort 낮은 채널
- 2순위 (준비 후 실행): Impact 높고 + Effort 높은 채널
- 3순위 (여유 시 실행): Impact 낮고 + Effort 낮은 채널
```

### 4. 주간 실행 체크리스트

```
- 화: {작업}
- 수: {작업}
- 목: {작업}
- 금: {작업/리뷰}
```
