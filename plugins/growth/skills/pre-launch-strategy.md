# Pre-launch Strategy

Step 5.2: 프리런칭 전략 수립

## 설명

출시 전 관심과 수요를 확보하기 위한 전략을 수립하는 스킬입니다. 타겟 시장(국내/글로벌)에 따라 적합한 채널과 전술을 선택합니다.

## 트리거

- Step 5.1 (Market Targeting) 완료 후

## 입력

- `outputs/stage-5/market-analysis.md` — 타겟 시장 및 페르소나

## 실행 내용

### 1. 웨이트리스트/랜딩페이지 전략

```
- 랜딩페이지 핵심 메시지 설계
- 이메일 수집 메커니즘 (LaunchList, Waitlister 등)
- 추천인 순위제 설계 (추천할수록 순위 상승 → 얼리 액세스)
- 스캐시티 메시지 활용 ("한정 초대", "VIP 얼리 액세스")
- 제품 관련 보상 설계 (추가 기능, 확장 용량 등)
```

### 2. 커뮤니티 빌딩

```
[국내]
- 카카오톡 오픈채팅 개설 (타겟 주제)
- 네이버 카페 활동 시작 (관련 커뮤니티)
- 디스콰이엇 프로필/제품 페이지 생성

[글로벌]
- Discord 또는 Slack 커뮤니티 생성
- Twitter/X "빌드 인 퍼블릭" 시작
- IndieHackers 프로필 생성, 여정 공유
```

### 3. 콘텐츠/SEO 전략

```
[국내]
- 네이버 블로그 개설 및 콘텐츠 계획 (1,500자+ 포스트)
- 네이버 SEO 최적화 (키워드, 태그, 참여도)
- Velog/브런치 기술 콘텐츠 발행

[글로벌]
- 블로그 + Google SEO (롱테일 키워드 중심)
- 비교 가이드, 튜토리얼 등 검색 의도 콘텐츠
- Dev.to / Medium 기고
```

### 4. 베타 테스팅 프로그램

```
- 웨이트리스트 상위 유저 → 베타 초대
- 피드백 수집 채널 설정 (인앱 / 폼 / 채팅)
- 피드백 반영 속도 극대화 (당일 반영 목표)
- 마이크로 인플루언서 베타 시딩
```

### 5. 정부 지원사업 (국내 전용)

```
- TIPS (최대 8억원 R&D 지원)
- 초기창업패키지 (최대 1.5억원)
- 예비창업패키지 (예비 창업자)
- 창업도약패키지 (3-7년차)
- 신청 포탈: k-startup.go.kr
- 신청 일정 확인 및 체크리스트 작성
```
