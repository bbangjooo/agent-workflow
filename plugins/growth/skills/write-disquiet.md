# Write Disquiet Launch

디스콰이엇 런칭 콘텐츠를 생성합니다.

## 설명

디스콰이엇(Disquiet)의 제품 런칭 포맷에 맞춘 콘텐츠를 생성합니다. 메이커 프로필, 제품 페이지, 런칭 포스트를 바로 올릴 수 있는 형태로 출력합니다.

## 입력

> **참고**: `outputs/promotion/copy-research.md`가 존재하면 경쟁사/성공 사례의 실제 카피 패턴을 참조하여 콘텐츠를 생성합니다. 검증된 패턴을 우리 제품에 맞게 변형합니다.

- 제품명, 한줄 설명, 핵심 기능, 타겟 유저, 라이브 URL, 메이커 소개

## 실행 내용

### 1. 제품 페이지

```
[제품명] — 한글 우선, 영문 병기 가능
[한줄 소개] — 30자 이내, 핵심 가치 전달
[상세 설명] — 한글 200-500자
  - 어떤 문제를 해결하는지
  - 핵심 기능 3-5개
  - 누구를 위한 것인지
  - 차별점

[카테고리] — 디스콰이엇 카테고리 선택
  (SaaS, 생산성, 개발자 도구, AI, 커뮤니티 등)

[태그] — 관련 태그 5-10개
```

### 2. 런칭 포스트 (메이커 로그)

```
구조:
- 인사 + 제품 소개 (1-2줄)
- 만든 배경 / 문제 경험 (3-5줄)
- 핵심 기능 설명 (불릿 3-5개)
- 기술적 디테일 (선택, 개발자 커뮤니티 어필)
- 앞으로의 계획 (1-2줄)
- 피드백 요청 (1줄)

톤: 한국어, 친근하지만 전문적
길이: 300-800자
이모지: 적절히 사용 (과하지 않게)
```

### 3. 메이커 프로필

```
- 자기소개 (1-2문장)
- 만들고 있는 것 / 관심 분야
- SNS 링크 (Twitter, GitHub 등)
```

### 4. 스크린샷 가이드

```
1. 메인 히어로: 제품 대표 화면
2. 핵심 기능: 3-4장
3. 한글 캡션 포함 권장
```

## 산출물

`outputs/promotion/disquiet/launch-content.md`

```markdown
# 디스콰이엇 런칭 콘텐츠

## 제품 페이지 정보

- **제품명**: {제품명}
- **한줄 소개**: {30자 이내}
- **카테고리**: {카테고리}
- **태그**: {태그 목록}
- **URL**: {라이브 URL}

## 상세 설명
{200-500자 상세 설명}

## 런칭 포스트 (메이커 로그)
{전체 본문 — 바로 복붙 가능}

## 메이커 프로필
- **소개**: {자기소개}
- **관심 분야**: {분야}

## 스크린샷 구성
1. {히어로 이미지 설명}
2. {기능 스크린샷 1}
3. {기능 스크린샷 2}
4. {기능 스크린샷 3}
```

## 완료 조건

- 제품 페이지 정보 완성
- 상세 설명 한글 200-500자
- 런칭 포스트 한글 300-800자
- 메이커 프로필 작성
