# Pricing Strategy

Step 1.2: 가격 전략 수립

## 설명

선택한 수익 모델에 맞는 가격 전략을 수립하고 티어를 설계하는 스킬입니다.

## 트리거

- Step 1.1 (Revenue Model Analysis) 완료 후

## 입력

- `outputs/monetization/revenue-model.md` — 선택된 수익 모델

## 실행 내용

### 질문 가이드

1. "경쟁사 가격을 알고 계신가요? 모르면 같이 찾아볼게요."
2. "초기에 무료로 시작할 생각인가요, 바로 유료로 갈 건가요?"
3. "타겟 유저의 지불 능력은 어느 정도일까요?"

### 가격 결정 프로세스

```
1. 가격 결정 방법론 선택:
   - 가치 기반: "이 문제 해결에 유저가 얼마를 지불할까?"
   - 경쟁사 기반: 시장 가격 ±20% 범위
   - 비용 기반: 원가 × (1 + 목표 마진)

2. 티어 설계 (3-4개 권장):
   - Good-Better-Best 프레임워크
   - 각 티어의 "와우 기능" 1개 이상
   - 중간 티어로 유도 (디코이 효과)

3. 프리미엄 경계 결정:
   - 무료에서 제공할 핵심 가치
   - 유료 전환 트리거 (사용량 한도, 기능 제한, 팀원 수)
   - "충분히 유용하지만 더 원하게 만드는" 경계선

4. 심리적 가격 전략 적용:
   - 앵커링, 디코이, 참 프라이싱
   - 연간 할인 (15-25%)

5. 가격 지역화:
   - PPP (구매력 평가) 기반 가격 차등
   - 한국 원화 / USD / EUR 가격 설정
```
