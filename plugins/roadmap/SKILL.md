# Roadmap - 프로젝트 로드맵 관리 플러그인

프로젝트의 목표를 정의하고, 목표 달성을 위한 ROADMAP을 생성하며, 현재 진행 위치를 파악하고, 모든 활동이 최초 목표에 실제로 기여하는지를 지속적으로 검증하는 플러그인입니다.

## 핵심 가치

솔로 파운더가 프로젝트를 진행하다 보면 방향을 잃기 쉽습니다. 이 플러그인은 세 가지 질문에 항상 답할 수 있게 해줍니다:

1. **어디로 가고 있는가?** → 명확한 목표와 ROADMAP
2. **지금 어디에 있는가?** → 진행 상황 추적
3. **제대로 가고 있는가?** → 목표 정렬 검증

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/roadmap` | 프로젝트 목표를 파악하고 ROADMAP을 생성하거나, 기존 ROADMAP을 조회합니다 |
| `/where` | ROADMAP 상의 현재 위치를 파악하고, 지금까지 한 일과 앞으로 할 일을 정리합니다 |
| `/align` | ROADMAP의 각 항목이 최초 목표에 실제로 기여하는지 검증하고, 방향 이탈을 감지합니다 |
| `/next` | 목표 확인 → 현재 위치 분석 → 다음 할 일을 한 번에 추천합니다 |

## 산출물

모든 산출물은 `outputs/roadmap/` 디렉토리에 저장됩니다:

- `goal.md` - 프로젝트 목표 정의서
- `roadmap.md` - 단계별 ROADMAP
- `progress.md` - 현재 진행 상황 리포트
- `alignment.md` - 목표 정렬 검증 리포트
- `next-action.md` - 다음 액션 추천 리포트

## 워크플로우

```
/roadmap → 목표 파악 → ROADMAP 생성
                ↓
/where  → 현재 위치 파악 → 진행 리포트
                ↓
/align  → 목표 정렬 검증 → 방향 수정 제안
                ↓
        (필요시 /roadmap으로 돌아가 ROADMAP 수정)

/next   → 위 전체를 한 번에 실행 → "지금 뭐 하면 돼?"에 답변
```
