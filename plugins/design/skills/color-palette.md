# Color Palette

Step 2.1: 색상 체계 결정

## 설명

아이디어와 기획 내용을 바탕으로 프로젝트에 적합한 색상 체계를 논의하고 결정하는 스킬입니다. 디자인 시스템의 핵심 요소인 색상을 초기에 확정함으로써 이후 모든 디자인 결정의 기초를 마련합니다.

## 트리거

- Step 2.0 (Platform Selection) 완료 후 실행
- `outputs/stage-2/platform-selection.md` 파일이 존재할 때

## 입력

- `outputs/stage-0/idea-brief.md`
- `outputs/stage-1/prd.md`
- `outputs/stage-2/platform-selection.md`

## 실행 내용

### 사전 작업

1. idea-brief.md에서 제품 비전과 핵심 가치 추출
2. prd.md에서 타겟 사용자 특성 파악
3. 제품 성격에 맞는 색상 팔레트 후보 준비

### 질문 가이드

1. **제품 성격과 색상**
   - "어떤 느낌을 주고 싶은 서비스인가요? (신뢰, 활기, 차분함, 전문성 등)"
   - "타겟 사용자층이 선호할 것 같은 색상 톤이 있나요?"

2. **브랜드 색상 확인**
   - "이미 정해진 브랜드 색상이 있나요? (로고, 기존 자료 등)"
   - "특별히 사용하고 싶거나 피하고 싶은 색상이 있나요?"

3. **경쟁사/참고 서비스 색상 분석**
   - "경쟁 서비스들은 주로 어떤 색상을 사용하나요?"
   - "차별화된 색상 전략이 필요할까요, 아니면 업계 관례를 따를까요?"

4. **색상 팔레트 제안**
   - 제품 성격에 맞는 2-3개 색상 팔레트 옵션 제시
   - 각 옵션의 장단점과 적합한 상황 설명

### 색상 심리학 가이드

| 색상 계열 | 연상 이미지 | 적합한 서비스 |
|----------|-------------|---------------|
| Blue (파랑) | 신뢰, 안정, 전문성, 차분함 | 금융, 헬스케어, 기업용, B2B |
| Green (초록) | 성장, 자연, 건강, 균형 | 웰니스, 친환경, 재정 관리 |
| Purple (보라) | 창의성, 고급스러움, 혁신 | 크리에이티브, 프리미엄 |
| Orange (주황) | 에너지, 친근함, 열정, 행동 유도 | 소셜, 이커머스, 스타트업 |
| Red (빨강) | 긴급함, 열정, 주목, 경고 | 미디어, 음식, CTA 버튼 |
| Yellow (노랑) | 낙관, 명랑, 따뜻함, 주의 | 어린이, 엔터테인먼트 |
| Teal (청록) | 현대적, 세련됨, 균형 | 테크 스타트업, 헬스테크 |
| Gray/Black | 세련됨, 중립, 프리미엄 | 럭셔리, 미니멀, 프로덕티비티 |

### 팔레트 구성 원칙

1. **Primary Color**: 브랜드를 대표하는 주 색상 (1개)
2. **Secondary Color**: 보조 역할을 하는 색상 (1-2개, 선택사항)
3. **Neutral Colors**: 텍스트, 배경, 경계선용 그레이스케일
4. **Semantic Colors**: 성공/경고/에러/정보를 위한 기능적 색상

### 대화 원칙

- 색상 심리학을 이해하기 쉽게 설명
- 구체적인 HEX 코드와 함께 제안
- 실제 서비스 예시를 들어 설명
- 사용자가 결정하기 쉽게 명확한 옵션 제시
- MVP에 필요한 최소한의 색상에 집중
