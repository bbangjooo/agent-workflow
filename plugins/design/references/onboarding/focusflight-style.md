# 온보딩 디자인 레퍼런스: FocusFlight 스타일

> 레퍼런스 URL: https://screensdesign.com/showcase/focusflight-deepfocus-timer
> 카테고리: 모바일 앱 온보딩, 테마형 몰입 경험, 생산성 앱

## 핵심 디자인 원칙

1. **일관된 테마 메타포** — 항공/여행 테마를 온보딩부터 핵심 기능까지 전면 적용 ("Airplane Mode" = 방해 차단, "Flight" = 집중 세션)
2. **다감각 몰입 환경** — 3D 글로브, 앰비언트 사운드, 비행기 창문 뷰로 시각+청각 경험 동시 제공
3. **인터랙티브 개인화** — 플로팅 태그 선택 등 사용자가 직접 참여하는 설정 단계
4. **보딩패스 은유** — 체크인/탑승권 컨셉으로 커밋먼트를 유도하는 탄젠트 UI
5. **테마 기반 기능 네이밍** — 기능명 자체가 항공 용어로 즉시 목적 파악 가능

## 온보딩 플로우 구조

```
[Splash] 앱 소개 + 비디오 배경 애니메이션
  ↓
[Value Prop] 핵심 가치 소개 (애니메이션 텍스트 + 비디오 배경)
  ↓
[Personalization] 플로팅 태그로 집중 시나리오 선택 (학습, 업무, 명상 등)
  ↓
[Boarding Pass] 탑승권 UI로 사용자 설정 요약 → 커밋먼트 유도
  ↓
[3D Globe] 목적지(집중 세션) 선택 — 맵 줌인 애니메이션
  ↓
[Feature Tour] 핵심 기능 소개 (Airplane Mode, Window View 등)
  ↓
[Paywall] 프리미엄 기능 캐러셀 ("Unlimited Flights", "-60% 할인")
  ↓
[Welcome] 구매 후 환영 화면 → 앱 평가 프롬프트 (긍정 감정 캡처)
  ↓
[Main Screen] 3D 글로브 메인 인터페이스
```

> 총 약 17단계. 테마 몰입에는 효과적이나 길이 최적화 여지 있음.

## 색상 팔레트

| 용도 | 색상 | 특성 |
|------|------|------|
| 배경 (기본) | 딥 네이비 / 다크 블루 | 야간 비행 분위기 |
| 배경 (글로브) | 다크 그라데이션 | 우주/하늘 연출 |
| 액센트 | 선셋 오렌지 / 골드 | 출발, CTA 강조 |
| 텍스트 (제목) | 화이트 | 고대비 가독성 |
| 텍스트 (본문) | 소프트 그레이 | 부드러운 보조 정보 |
| 태그 (선택 전) | 반투명 화이트 | 플로팅 효과 |
| 태그 (선택 후) | 오렌지 / 골드 하이라이트 | 선택 피드백 |

## 타이포그래피

```
헤딩: 대형 볼드, 애니메이션 텍스트 등장 효과
본문: 중간 사이즈, 충분한 행간으로 가독성 확보
라벨: 소형, 항공 테마 용어 (Gate, Seat, Destination 등)
보딩패스: 모노스페이스 혼합 (항공권 느낌)
```

## 인터랙션 & 애니메이션 패턴

### 보딩패스 체크인 스크롤
```
세로 스크롤 → 보딩패스가 점진적으로 나타남
체크인 완료 시 → 티어링(찢기) 애니메이션으로 탑승 전환
```

### 3D 글로브 목적지 선택
```
메인 화면 중앙에 3D 글로브 렌더링
터치/스와이프로 글로브 회전 → 목적지 핀 선택
선택 시 맵 줌인 애니메이션 → 세션 시작 화면 전환
```

### 플로팅 태그 선택
```
집중 시나리오 태그들이 화면에 부유하며 표시
탭하면 태그가 강조(색상 변경 + 스케일 업)
여러 개 선택 가능 → 개인화 데이터 수집
```

### Duration 슬라이더
```
제스처 기반 스와이프로 집중 시간 조절
연속적 피드백 + 햅틱 진동 (분 단위)
```

### 온보딩 시퀀스 스케일 백
```
화면 전환 시 이전 화면이 스케일 다운 → 뒤로 밀려남
새 화면이 전면으로 스케일 업 → 깊이감 연출
```

## 수익화 통합 패턴

```
[Paywall 시점] 온보딩 중반 (기능 소개 직후)
[형식] 프리미엄 기능 캐러셀 슬라이드
[강조] 연간 플랜 -60% 할인 + 월 환산 가격 표시
[테마 유지] "Unlimited Flights" 등 항공 용어 사용
[후속] 구매 완료 → 환영 축하 화면 → 앱 평가 요청 (긍정 감정 시점)
```

## 디자인 톤앤무드

**몰입형 테마 경험 + 감성적 생산성**
- 항공 메타포를 UI 전반에 일관 적용하여 세계관 구축
- 다감각 요소(시각 + 청각)로 집중 환경 연출
- 보딩패스, 찢기 애니메이션 등 탄젠트 인터랙션으로 재미 요소
- 긴 온보딩이지만 각 단계가 인터랙티브하여 이탈 방지
- 기능명이 곧 테마 용어 → 학습 비용 감소

## 참고 포인트

- **테마 메타포 온보딩** — 앱의 핵심 컨셉을 온보딩 전 과정에 녹여내는 방법
- **인터랙티브 개인화** — 플로팅 태그 등 참여형 설정으로 사용자 관여도 향상
- **보딩패스 UI 패턴** — 커밋먼트 유도 + 설정 요약을 하나의 은유로 해결
- **3D 글로브 인터페이스** — 공간적 메타포로 세션 선택을 직관화
- **수익화 타이밍** — 기능 투어 직후 → 가치 인지 최대 시점에 paywall 제시
- **감정 활용 평가 요청** — 구매 직후 긍정 감정 시점에 앱 평가 프롬프트
- **긴 온보딩의 트레이드오프** — 테마 몰입 vs 첫 핵심 경험까지의 거리
